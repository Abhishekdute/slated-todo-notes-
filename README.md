# 🖋️ Slated - Premium Task & Notes App

**Slated** is a modern, feature-rich productivity tool built with Flutter. It combines a powerful Todo list with a creative Notes taking experience, featuring multimedia attachments and a polished UI.

## ✨ Key Features

- **🚀 Task Management**: Create, update, and manage your daily tasks with ease.
- **📝 Creative Notes**: Staggered grid view for notes with custom background colors.
- **🎨 Multimedia Attachments**:
  - 📷 Capture or pick images from the gallery.
  - ✍️ Handwriting support via a custom Drawing Pad.
- **⏰ Reminders**: Set alarms and reminders for your important tasks.
- **🌙 Dark Mode**: Fully responsive UI with a premium dark theme support.
- **🛡️ Local Storage**: Uses **Hive** for lightning-fast local data persistence.
- **⚡ Smooth Animations**: Powered by `animate_do` for a high-end feel.

## 🛠️ Tech Stack

- **Framework**: Flutter
- **State Management**: Provider
- **Database**: Hive
- **UI Components**: 
  - `flutter_slider_drawer`
  - `flutter_staggered_grid_view`
  - `lottie` animations
